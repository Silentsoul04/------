URL 请求时会对百分号等 URL 的编码方式进行转码；
浏览器接收到页面数据后，会对 HTML 实体编码进行转码；
执行 JS 时会对 JSUnicode 等 JS 支持的方式进行转码。

HTML 中有用的字符实体
注释：实体名称对大小写敏感！

|显示结果	|描述	|实体名称	|实体编号|
|-----------|-------|-----------|-------|
|			|空格	|&nbsp;		|&#160;
|<			|小于号	|&lt;		|&#60;
>			|大于号	|&gt;		|&#62;
&			|和号	|&amp;		|&#38;
"			|引号	|&quot;		|&#34;
'			|撇号 	|&apos; (IE不支持)	|&#39;
