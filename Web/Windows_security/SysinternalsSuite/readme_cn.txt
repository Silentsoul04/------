Sysinternals故障排除实用程序已被汇总为一套工具。该文件包含单独的故障排除工具和帮助文件。它不包含非故障排除工具，如BSOD屏幕保护程序或NotMyFault。
有关Sysinternals和这些实用程序的更多信息，请访问网站 - http://technet.microsoft.com/sysinternals

该套件包含以下选定的Sysinternals实用程序：


AccessChk - AccessChk是一个命令行工具，用于查看文件，注册表项，服务，进程，内核对象等的有效权限。

AccessEnum - 这个简单但功能强大的安全工具显示谁可以访问系统上的目录，文件和注册表项。使用它来查找权限中的漏洞。

AdExplorer - Active Directory Explorer是高级Active Directory（AD）查看器和编辑器。

AdInsight - 旨在解决Active Directory客户端应用程序故障的LDAP（轻量级目录访问协议）实时监控工具。

AdRestore - 取消删除Server 2003 Active Directory对象。

自动登录 - 登录时绕过密码屏幕。

自动运行 - 查看当您的系统启动并登录时，哪些程序配置为自动启动。 Autoruns还会显示应用程序可以配置自动启动设置的注册表和文件位置的完整列表。

BgInfo - 这个完全可配置的程序自动生成桌面背景，其中包括有关系统的重要信息，包括IP地址，计算机名称，网络适配器等。

CacheSet - CacheSet是一个允许您使用NT提供的函数来控制缓存管理器的工作集大小的程序。它与所有版本的NT兼容。

ClockRes - 查看系统时钟的分辨率，这也是最大的定时器分辨率。

Contig - 希望您能快速对经常使用的文件进行碎片整理吗？使用Contig优化单个文件，或创建连续的新文件。

Coreinfo - Coreinfo是一个新的命令行实用程序，向您显示逻辑处理器与其所在的物理处理器，NUMA节点和套接字之间的映射以及分配给每个逻辑处理器的缓存。

Ctrl2cap - 这是一个内核模式驱动程序，它在键盘类驱动程序上方演示键盘输入过滤，以便将大写锁定转换为控制键。在这个级别过滤允许在NT之前转换和隐藏密钥甚至“看见”它们。 Ctrl2cap还显示了如何使用NtDisplayString（）将消息打印到初始化蓝屏。

DebugView - 来自Sysinternals的另一个第一：该程序拦截由设备驱动程序和Win32程序制作的OutputDebugString对DbgPrint的调用。它允许在本地机器上或通过互联网查看和记录调试会话输出，而无需使用主动调试器。

桌面 - 这个新的实用程序使您能够创建最多四个虚拟桌面，并使用托盘界面或热键预览每个桌面上的内容并轻松地在它们之间切换。

Disk2vhd - Disk2vhd简化了物理系统向虚拟机（p2v）的迁移。

DiskExt - 显示卷磁盘映射。

Diskmon - 该实用程序可捕获所有硬盘活动，或者像系统托盘中的软件磁盘活动指示灯一样。

DiskView - 图形磁盘扇区实用程序。

磁盘使用（DU） - 按目录查看磁盘使用情况。

EFSDump - 查看加密文件的信息。

句柄 - 这个方便的命令行实用程序将向您显示哪些文件由哪些进程打开，等等。

Hex2dec - 将十六进制数转换为十进制数，反之亦然。

连接点 - 创建Win2K NTFS符号链接。

LDMDump - 转储逻辑磁盘管理器的磁盘数据库的内容，该数据库描述了Windows 2000动态磁盘的分区。

ListDLLs - 列出当前加载的所有DLL，包括它们的加载位置及其版本号。版本2.0打印加载模块的完整路径名称。

LiveKd - 使用Microsoft内核调试程序检查实时系统。

LoadOrder - 查看WinNT / 2K系统中加载设备的顺序。

LogonSessions - 列出系统上的活动登录会话。

MoveFile - 允许您为下次重新启动安排移动和删除命令。

NTFSInfo - 使用NTFSInfo查看有关NTFS卷的详细信息，包括主文件表（MFT）和MFT区的大小和位置，以及NTFS元数据文件的大小。

PageDefrag - 对页面文件和注册表配置单元进行碎片整理。

PendMoves - 列举将在下次启动时执行的文件重命名和删除命令列表。

PipeList - 显示系统上的命名管道，包括每个管道的最大实例数和活动实例数。

PortMon-使用此高级监控工具监控串行和并行端口活动。它了解所有标准串行和并行IOCTL，甚至可以显示发送和接收的部分数据。 3.x版具有强大的新UI增强功能和高级过滤功能。

ProcDump - 这个新的命令行实用程序旨在捕获难以隔离和重现CPU峰值的进程转储。它还可用作通用进程转储创建实用程序，还可以在进程具有挂起窗口或未处理的异常时监视和生成进程转储。

Process Explorer - 找出哪些文件，注册表项和其他对象进程已打开，它们加载了哪些DLL等等。这个独一无二的强大工具甚至可以告诉你谁拥有每一个过程。

进程监视器 - 实时监视文件系统，注册表，进程，线程和DLL活动。

ProcFeatures - 此applet报告处理器和Windows对物理地址扩展和无执行缓冲区溢出保护的支持。

PsExec - 在远程系统上执行进程。

PsFile - 查看远程打开的文件。

PsGetSid - 显示计算机或用户的SID。

PsInfo - 获取有关系统的信息。

PsKill - 终止本地或远程进程。

PsList - 显示有关进程和线程的信息。

PsLoggedOn - 显示登录到系统的用户。

PsLogList - 转储事件日志记录。

PsPasswd - 更改帐户密码。

PsService - 查看和控制服务。

PsShutdown - 关闭并可选择重启计算机。

PsSuspend - 暂停和恢复进程。

RAMMap - 一种高级物理内存使用分析实用程序，可在其多个不同选项卡上以不同方式显示使用情况信息。

RegDelNull - 扫描并删除包含嵌入空字符的注册表项，这些空字符是标准注册表编辑工具无法删除的。

RegJump - 跳转到您在Regedit中指定的注册表路径。

RootkitRevealer - 扫描您的系统以查找基于rootkit的恶意软件。

SDelete - 使用符合DoD标准的安全删除程序，安全地覆盖敏感文件并清除以前删除文件的空闲空间。

ShareEnum - 扫描网络上的文件共享并查看其安全设置以关闭安全漏洞。

ShellRunas - 通过方便的shell上下文菜单条目以不同的用户身份启动程序。

Sigcheck - 转储文件版本信息并验证系统上的图像是否经过数字签名。

数据流 - 显示NTFS备用数据流。

字符串 - 在binaryimages中搜索ANSI和UNICODE字符串。

同步 - 将缓存的数据刷新到磁盘。

TCPView - 活动套接字命令行查看器。

VMMap - VMMap是一个进程虚拟和物理内存分析工具。

VolumeId - 设置FAT或NTFS驱动器的卷ID。

Whois - 查看谁拥有互联网地址。

WinObj - 最终的对象管理器命名空间查看器在这里。

ZoomIt - 用于在屏幕上缩放和绘图的演示实用程序。
